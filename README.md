# earth-colours-mapper — Earth Colours Mapper

An interactive map for exploring earth colours by geographic location.
Browse named colours by place, view HSV and luminance data, and generate colour palettes and cartographic gradients.

**[Live →](https://www.jonathancinnamon.com/earth-colours-mapper/)**

---

## Features

- Interactive map interface linking colour to geographic location
- Sortable colour table with hex, HSV, luminance, and coordinate data
- Filter and search by colour name, type, or place
- Create Earth colour palettes
- Generate colour ramps for mapping

## Usage

Browse colours on the map or in the table. Click a colour name or symbol to highlight
the location. Use the filter or search box to narrow the palette.
No installation or account required.
