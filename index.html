Create an Earth Colour Palette

Create a colour palette using named geographic colours. Hover any table row and click + to add a colour. Or click a map marker, then use + Add to palette in the info panel. The tray opens automatically along the bottom of the screen.

Drag chips to reorder. Use the Sort buttons to arrange by hue, saturation, brightness, or luminance. Hover a chip to reveal the × remove button. Clear empties the palette.

Cartographic Colour Ramp Generator

Create map colour gradients. Add desired colours to palette, at least 2 for a sequential colour scheme (e.g. 'high' to 'low'), 3 for diverging (e.g. 'hot', 'cold', 'neutral'). The ramp can interpolate additional classes through the CIELAB colour space, which matches human perception well for cartographic gradients.
Choose 3–11 classes, export as Hex/RGB/HSV for any software, PNG for image use, QGIS XML to import directly as a QGIS colour ramp, or CSS gradient for web maps.

Table & Filtering

Click any column header to sort — click again to reverse. Use the search box to filter by colour name, place, or hex value. The filter chips narrow results by geographic category (city, region, country, water, landform).
The ↕ drag handle between map and table resizes both panes. ↓ CSV (header) exports the current filtered and sorted view.

Map Interaction

Click any map marker to see the colour's name, hex code, and HSV values in the info panel (bottom-left). Use + Add to palette there to add it without leaving the map.

Clicking a table row pans and zooms the map to that location and highlights the marker. Zoom and pan normally with scroll and drag. Switch to Dark mode for a dark basemap.

Sources

Colour names and hex values are compiled from A Dictionary of Color by Maerz & Paul (McGraw-Hill, 1930); Methuen Handbook of Colour by Kornerup & Wanscher (3rd ed., 1978); Color Standards and Color Nomenclature by Robert Ridgway (1912); Winsor & Newton historical pigment catalogues; Werner’s Nomenclature of Colours (1814); the W3C CSS Color Module Level 4 specification; and the broader art-historical literature on named pigments and dyes. This is a curated rather than definitive list of named geographic colours. Geographic coordinates are approximate centroids from OpenStreetMap / Natural Earth.

Methodology & notes

Hex values are representative sRGB approximations — historical pigments exist in varied formulations and the values here reflect common modern reference renderings, not a single authoritative standard. Archaic, trade, and colonial-era designations carry historical and cultural complexity; their inclusion is documentary, not an endorsement of the naming conventions of any period. HSV values are computed from the hex representations. Some geographic locations appear more than once where distinct pigment formulations are documented.
